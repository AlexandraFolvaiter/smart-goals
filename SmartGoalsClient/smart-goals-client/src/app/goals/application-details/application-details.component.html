Application business details

Goal = the object of a person's ambition or effort; an aim or desired result.

Objective = a thing aimed at or sought; a goal. Statements describing the results to be achieved, and the manner in which they will be achieved. You usually need multiple objectives to address a single goal.

So you have a goal. And for the goal that you have you need to set objectives. Each objective should be S.M.A.R.T.:

  • Specific: includes the “who”, “what”, and “where”. Use only one action verb to avoid issues with measuring success.
  • Measurable: focuses on “how much” change is expected.
  • Achievable: realistic given program resources and planned implementation.
  • Relevant: relates directly to program/activity goals.
  • Time-bound: focuses on “when” the objective will be achieved.
